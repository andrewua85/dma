### Текущий рабочий вариант проекта (по состоянию на 08.03.2025)

#### Структура интерфейса
1. **Блок 0**: Переключение темы и языка (фиксированный сверху).
2. **Блок 1**: Поля ввода для формул с прокруткой:
   - Адаптивный по ширине: растягивается на всю ширину окна.
   - В режиме >400px: два столбца, поля ввода "резиновые" (без фиксированной ширины, растягиваются с помощью `sticky="ew"`).
   - В режиме ≤400px: одна колонка, поля ввода с фиксированной шириной (`width=20`).
3. **Блок 2**: Кнопки "Рассчитать" и "Очистить" (фиксированный, не прокручивается).
4. **Блок 3**: Результаты (растягивается по высоте, адаптивный).
5. **Блок 4**: Кнопки "Сохранить в файл", "Загрузить из файла" и статус:
   - Фиксированная высота 50px, всегда виден.

#### Файлы проекта
1. **`src/ui.py`**:
   - Содержит класс `AppUI` с полной реализацией интерфейса и логики (последняя версия из предыдущего сообщения).
   - Адаптивность блока 1 реализована через `configure_canvas` и `grid_columnconfigure`.

2. **`src/utils.py`**:
   - Класс `ToolTip` для подсказок.
   - Функции `save_data` и `load_data` для работы с `metrics_data.json`.
   - Функция `get_base_path` для определения пути сохранения (рядом с `.exe` при компиляции).

3. **`src/main.py`** (предположительно, без изменений):
   ```python
   import tkinter as tk
   import logging
   from src.ui import AppUI

   logging.basicConfig(level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')
   logging.info("Запуск приложения")

   if __name__ == "__main__":
       root = tk.Tk()
       app = AppUI(root)
       root.mainloop()
   ```

4. **`src/calculations.py`** и **`src/charts.py`**:
   - Предполагаю, что они остались без изменений и работают корректно (расчёты и графики).

#### Функционал
- **Сохранение/загрузка**: Данные сохраняются в `metrics_data.json` в корневой папке проекта (или рядом с `.exe`).
- **Переключение темы и языка**: Работает корректно.
- **Расчёты**: Выполняются при нажатии "Рассчитать".
- **Очистка**: Сбрасывает все поля и результаты.

#### Команды
- Запуск: `python src/main.py`
- Компиляция: `pyinstaller --onefile --add-data "assets;assets" src/main.py`
